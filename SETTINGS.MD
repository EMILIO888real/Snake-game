# Settings

1. **name**: Player/game profile name (default: `"Joker"`)
2. **screen size**: Display resolution in pixels [width, height] (default: `[640, 480]`)
3. **ups**: Updates per second (default: `240`)
4. **fps**: Frames per second (default: `240`)
5. **color pattern length**: Number of colors in the pattern cycle (default: `5`)
6. **color scheme**: Predefined color theme name (default: `"neon_cyberpunk"`)
7. **color style**: Color theme variant selector (default: `1`)
8. **portals**: Enable/disable portal wrapping (default: `true`)
9. **eating speeds you up**: Whether consuming food increases speed (default: `false`)
10. **eating speed you up amount**: Speed increase per food consumed (default: `10`)
11. **audio**: Master audio toggle (default: `true`)
12. **music**: Enable/disable background music (default: `true`)
13. **sfx**: Enable/disable sound effects (default: `true`)
14. **master volume**: Master audio volume 0.0-1.0 (default: `1.0`)
15. **music volume**: Background music volume 0.0-1.0 (default: `0.2`)
16. **eating sfx volume**: Eating sound volume 0.0-1.0 (default: `0.3`)
17. **lose sfx volume**: Loss sound volume 0.0-1.0 (default: `0.5`)
18. **music name**: Background music track selection (default: `"random"`)
19. **fullscreen**: Enable/disable fullscreen mode (default: `false`)
20. **snakes count**: Number of snakes in the game (default: `1`)
21. **grid lines**: Enable/disable grid lines (default: `true`)
22. **grid lines color**: RGB color for grid lines (default: `[40, 40, 40]`)
23. **cycle food colors**: Enable/disable cycling food colors (default: `true`)
24. **last piece becomes food color**: Whether the last piece becomes the food color (default: `true`)
25. **food starting color**: Starting color index for food (default: `1`)
26. **snake starting color**: Starting color index for snake (default: `0`)
27. **playlist**: Enable/disable music playlist (default: `true`)
28. **sequential playlist**: Enable/disable sequential playlist (default: `false`)
29. **careful between snakes collision detection**: Enable/disable careful collision detection between snakes (default: `false`)
30. **borderless**: Enable/disable borderless window mode (default: `false`)
31. **show performance stats**: Enable/disable performance stats display (default: `false`)
32. **color performance stats**: Enable/disable colored performance stats (default: `true`)
33. **percentage volume notification**: Enable/disable percentage volume notification (default: `true`)
34. **use image assets**: Enable/disable use of image assets (default: `false`)
35. **move up key**: Key for moving up (default: `"w"`)
36. **move down key**: Key for moving down (default: `"s"`)
37. **move left key**: Key for moving left (default: `"a"`)
38. **move right key**: Key for moving right (default: `"d"`)
39. **quit game key**: Key for quitting the game (default: `"q"`)
40. **toggle grid lines key**: Key for toggling grid lines (default: `"g"`)
41. **toggle performance stats key**: Key for toggling performance stats (default: "`")
42. **pause key**: Keys for pausing/unpausing the game (default: `["escape", "space"]`)
43. **increase volume key**: Key for increasing volume (default: `"kp_plus"`)
44. **decrease volume key**: Key for decreasing volume (default: `"kp_minus"`)
45. **forward music key**: Key for forwarding music track (default: `"f"`)
46. **backward music key**: Key for backward music track (default: `"b"`)
47. **repeat current song key**: Key for repeating current song (default: `"r"`)
48. **repeat current song modifier key**: Modifier key for repeat (default: `"lalt"`)
49. **safe food spawn**: Enable/disable safe food spawning (default: `true`)
50. **slower key inputs**: Enable/disable slower key inputs (default: `true`)
51. **restart game key**: Key for restarting the game (default: `"y"`)
52. **compatibility**: Enable/disable compatibility mode (default: `true`)
53. **show time**: Enable/disable time display (default: `true`)
54. **resizable window**: Enable/disable resizable window (default: `false`)
55. **toggle stopwatch key**: Key for toggling stopwatch (default: `"u"`)
56. **GUI**: Enable/disable GUI (default: `true`)
57. **disable soft restart key**: Key for disabling soft restart (default: `"home"`)
58. **crash game key**: Key for crashing the game (default: `"end"`)
